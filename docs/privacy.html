Privacy Policy

Effective Date: May 2025

Overview

Draw On Screen is a Chrome extension that allows users to draw directly on webpages, scroll pages while keeping annotations, and capture screenshots of their work. The extension is designed for a single purpose: enabling visual annotations on live websites.

Data Collection

This extension does not collect, transmit, or store any personal data. All annotations and settings are handled locally in the browser and are not sent to any server.

Permissions Justification

  • activeTab – required to activate drawing functionality on the current webpage when the extension icon is clicked.
  • scripting – needed to inject scripts that create and manage canvas elements on the webpage, enabling drawing tools.
  • downloads – necessary to allow users to save screenshots of their annotations to their device.

Remote Code

This extension does not use remote code. All JavaScript, CSS, and assets are packaged within the extension and loaded locally. There are no external <script> references or eval()-based code execution.

User Data

Draw On Screen does not collect any user data. Specifically, it does not access, collect, or process:

  • Personally identifiable information
  • Health or financial information
  • Authentication data or credentials
  • Communications or messages
  • Location or IP address
  • Browsing history
  • User activity (such as keystrokes or mouse tracking)
  • Website content

Data Usage Commitments

We certify the following:

  • We do not sell or transfer user data to third parties, apart from approved use cases (not applicable in this case).
  • We do not use or transfer user data for purposes unrelated to the extension’s single purpose.
  • We do not use or transfer user data to determine creditworthiness or for lending purposes.

Changes

This privacy policy may be updated in the future. All changes will be published on this page.

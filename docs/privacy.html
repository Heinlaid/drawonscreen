Privacy Policy

Effective Date: May 2025

Overview

Draw On Screen is a Chrome extension that lets users draw, highlight, and annotate directly on webpages and capture screenshots. We are committed to protecting your privacy and being transparent about data handling.

Data Collection

This extension does not collect, transmit, or store any personal data, browsing history, or website content externally. All drawings and user data remain in your browser’s storage (chrome.storage.local).

Permissions

The extension requests the following Chrome permissions:

  • activeTab — inject drawing tools into the current webpage.
  • storage — save tool settings and drawings in your browser.
  • clipboardWrite — copy screenshots of annotated pages to clipboard.

None of these permissions are used to collect or transmit personal data.

Remote Code

The extension does not execute remote code. All JavaScript and assets are packaged with the extension.

Changes

We may update this Privacy Policy. Updates will be posted here.
